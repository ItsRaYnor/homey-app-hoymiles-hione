Local connection (LAN)

  

    The IP address of the DTS-G3 stick on your network. Saved here, it is prefilled when pairing.
  

  
  
  
  
  
For the DTS-G3 use Modbus TCP on port 502. The Modbus unit ID is usually in the 101–254 range. First enable "Remote Control" on the stick (see below).

  
  


  

    Enable Modbus on the DTS-G3: open the S-Miles Installer app →
    Me → Local Install Assistant / Toolkit → DTU Information → RS485 Port Config →
    choose Remote Control (NOT Export Control — that blocks Modbus).
    Set the Modbus/RS485 address in the 101–254 range.
